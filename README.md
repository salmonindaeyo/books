# ระบบยืมหนังสือแบบคร่าวๆ (5-6hr)

## Tech Stack

**Client:** React, Zustand, TailwindCSS , react query , Mui UI

**Server:** Node, Express , prisma

# Set up project BE

```javascript
cd be
```

### run db

```javascript
make up
```

or

```javascript
docker-compose up -d
```

### run server

```javascript
bun i or npm i or yarn i
```

```javascript
npx prisma migrate reset
```

```javascript
bun run dev .... npm run dev
```
